Pour lancer le programme lancez votre cmd
mettez vous bien dans le dossier src du jeu
Puis faites la commande suivante : "mingw32-make"
Une fois le programme compilé
Faites Bomberman2D.exe

Si toute fois votre compilation ne marche pas avec les fichiers de github téléchargez les fichiers
sur google drive avec ce lien et compiler le sur windows avec mingw32 :  
https://drive.google.com/file/d/1JuA6Ap8K2_zclZZZrU4Zag9MKVcr3FS3/view?usp=sharing


Voici toutes les commandes à connaitre pour chacun des joueurs

Pour le joueur 1(rouge) 
	a : poser une bombe
	e : rotation du joueur
	z : avancer vers le haut
	q : avancer vers la gauche
	s : avancer vers le bas
	d : avancer vers la droite

Pour le joueur 2(bleu) 
	r : poser une bombe
	y : rotation du joueur
	t : avancer vers le haut
	f : avancer vers la gauche
	g : avancer vers le bas
	h : avancer vers la droite

Pour le joueur 3(jaune) 
	u : poser une bombe
	o : rotation du joueur
	i : avancer vers le haut
	j : avancer vers la gauche
	k : avancer vers le bas
	l : avancer vers la droite

Pour le joueur 4(vert) 
	ctrl droit : poser une bombe
	maj droit : rotation du joueur
	flèche du haut : avancer vers le haut
	flèche de gauche : avancer vers la gauche
	flèche du bas : avancer vers le bas
	flèche de droite : avancer vers la droite
